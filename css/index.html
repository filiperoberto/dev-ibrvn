CSS

O que é?

  • Cascading Style Sheets
  • Linguagem para dar estilo ao HTML
seletor { 
    propriedade: valor;
}

Como aparece?

  • Inline
  • Interno
  • Externo

CSS Inline

<h1 style='background: #fbfb73;'>My First Heading</h1>

CSS Interno

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
<style> 
h1 { 
    background: #fbfb73; 
} 
</style>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

CSS Externo

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
<link rel="stylesheet" href="style.css" />
</head>
<body>
...

Seletores

Uma parte importante ao aplica o CSS é selecionar qual elemento/tag queremos aplicar o estilo. Os Seletores servem como uma forma de "busca" por um ou mais elementos específicos em meio a todas tags do documento HTML.

Tipos de seletores

  • Tag
  • Id
  • Classe
  • Outros

Tag

li { color:red; }
<ul>
    <li>Sol</li>
    <li>Mercúrio</li>
    <li>Vênus</li>
    <li>Terra</li>
    <li>Marte</li>
</ul>

Id

#terra { color:red; }
<ul>
    <li>Sol</li>
    <li>Mercúrio</li>
    <li>Vênus</li>
    <li id='terra'>Terra</li>
    <li>Marte</li>
</ul>

Classe

.planeta { color:red; }
<ul>
    <li class='estrela'>Sol</li>
    <li class='planeta'>Mercúrio</li>
    <li class='planeta'>Vênus</li>
    <li class='planeta' id='terra'>Terra</li>
    <li class='planeta'>Marte</li>
</ul>

Outros

  • Pseudo elementos
  • Atributo
