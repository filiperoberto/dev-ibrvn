Pré introdução

Uma das questões que nos deparamos no desenvolvimento web é a "massaroca" de tecnologias que estão envolvidas. Já vi muita gente entrando nesse meio e se confundindo, não sabendo diferenciar direito quando está mexendo com HTML ou com javascript ou com CSS ou com alguma biblioteca envolvida.

Por isso achei por bem explicar um pouco de cada um separadamento para que tenhamos a visão de como um interage com o outro, e quando precisarmos, vamos saber melhor com o que estamos lidando, se é questão de mexer no HTML, ou no CSS e etc.

Introdução ao HTML

O que é?

HTML é uma linguagem de marcação. De forma semelhante a quando criamos um documento no word por exemplo, onde existem títulos, parágrafos, listas, citações, tabelas e etc. Também o HTML nos permite criar documentos e formatá-los da maneira que precisarmos.

Ilustração do corpo humano

Estrutura básica do HTML

  <!DOCTYPE html>
  <html>
  <head>
  <title>Page Title</title>
  </head>
  <body>
  
  <h1>My First Heading</h1>
  <p>My first paragraph.</p>
  
  </body>
  </html>

Estrutura das tags

* Os atributos class e id serão importantes depois para aplicar CSS e javascript

O HTML possui:

  • Tags visíveis
  • Tags não visíveis
    • Metadados
    • Elementos escondidos
    • Tags de importação de CSS/javascript

O Navegador já aplica um estilo padrão para alguns elementos, mesmo sem CSS

A tag strong por exemplo, aplica negrito, as tags de h (h1, h2, h3 ...) aplicam uma margem e tamanho de fonte diferente ao texto.

Posicionamento:

Uma questão importante no HTML e depois no CSS, é o posicionamento. Como posicionar um elemento em determinado lugar na tela? Duas formas de posicionamento importantes no HTML são inline e block.
A div é um elemento HTML
que por padrão
Tem posiciomanemto block
Ou seja, ele "bloqueia" a linha onde ele está
Não permitindo que outros elementos sejam posicionanos
Na mesma linha
Por outro lado o span é um elemento inline, eu posso colocar vários elementos inline e eles ficarão posicionados na mesma linha, a não ser que o tamanho da linha acabe neste caso, o elemento será jogado para baixo

Na página do w3schools tem uma lista de elementos que por padrão são block ou inline

Interação com elementos

É possível o usuário interagir com algumas tags html. Ou seja, elas possuem funcionalidade especial, não somente de formatação de texto.

Por exemplo a tag <a> cria um link com outra página.
